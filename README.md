# The Real Speed Test (EU)

A lightweight Windows app focused on Lebanon users:
- **Cache speed** (local/CDN/peered)
- **Real EU speed** (actual EU throughput)
- **EU city ping** (Frankfurt / Paris / Amsterdam / London)
- **Jitter + packet loss**
- Simple **gaming verdict** (Good / Okay / Bad)

## Download
Go to **Releases** and download the latest ZIP.

## How to run
1. Download the ZIP from Releases
2. Extract it anywhere
3. Run `TheRealSpeedTest.exe`
